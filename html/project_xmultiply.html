X-Multiply Tribute
Category: Videogame
Year: 2017
Team Size: 4

X-Multiply Tribute was the first videogame I have ever made.

The game is a side-scrolling shooter game where the player controls a microscopic fighter ship that aims to end with an aliens plague. We developed the Stage-5 of the game.

The tribute is programmed in C++ using SDL library and PNG sprites.
