Jour
Category: Mobile App
Year: 2023
Team Size: Myself

At the start of each academic year, students search for the perfect tool that helps them to organize their studies.

However, it's difficult to find an all-in-one productive tool that is not too complex to learn or it's incomplete if the premium option isn't purchased. This application would help students to plan their time to be more productive. Besides, it is simple to learn, intuitive and free.

Jour is an intuitive mobile application design. It has been deleveloped in Figma for a UX Case Study. There is an interactive prototype for the app as well as a case study presentation.
