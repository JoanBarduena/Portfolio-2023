The Witcher: Ties of Destiny
Category: Videogame
Year: 2020
Team Size: 29

The Witcher: Ties of Destiny is a 3D Hack and Slash with fast pacing mechanics which takes place in The Witcher universe.

The game was developed with Broken Engine (our own open-source 3D game engine built from the ground with C++ and OpenGL).

The game website was developed by Xavi Marin and I, which took us more than 300 hours. It was created from a template and programmed with CSS and HTML5.
