Portfolio Website
Category: 3D Art
Year: 2022
Copies sold: <100

This website has been created from scratch without any template. It is done with HTML, CSS and Javascript.

The website has some features coded with Javascript:

  • Theme mode (dark/light). The theme is stored in the browser so whenever the user enters again in the website it loads the selected theme.
  • Project year filter. The project years are displayed from oldest to newest year.
  • Cursor. Custom cursor created with GSAP animations.
  • Introduction change text animation.
  • Back button goes to previous opened project.
  • Navigation filter on projects.
