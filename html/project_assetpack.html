Stylized Medieval Village
Category: 3D Art
Year: 2022
Copies sold: <100

This stylized medieval village asset pack was my Bachelor's Thesis. It contains a set of stylized modular walls, roofs, doors, windows, stairs and street decorations, so the user can create their customized environment.

I created every single mesh and texture from scratch.

I used Blender (3D Model), Adobe Substance Designer (tileable textures), Adobe Substance Painter (singular prop textures) and Unreal Engine 4.
