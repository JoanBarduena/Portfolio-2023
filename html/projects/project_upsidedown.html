Upside Down
Category: Videogame
Year: 2018
Team Size: 2

Upside down is a 2D platformer video game that has been made using C++ (with SDL and PUGI libraries) and two maps created with Tiled.

The game consists on reaching the end of level 2 picking the maximum amount of coins while not dying.

The main feature involves swapping gravity upside down to avoid enemies and move along the level.
